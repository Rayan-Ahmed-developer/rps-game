✊✋✌ Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. This project is great for beginners to understand basic game logic, DOM manipulation, and event handling.

🧩 Features
🎮 Play against the computer
⚡ Instant result display (Win / Lose / Draw)
🧠 Random computer moves
📊 Score tracking system
🔄 Play again / restart option
🎨 Simple and responsive UI

🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)

📁 Project Structure
Rock-Paper-Scissor-Game/
│
├── index.html
├── style.css
├── script.js
└── README.md

🎮 How to Play
Choose one option:
Rock ✊
Paper ✋
Scissors ✌
The computer randomly selects its move
The result is displayed instantly
Score updates automatically after each round
